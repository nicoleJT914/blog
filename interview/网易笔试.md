2. setTimeout()
`var timerId = setTimeout(func, delay)`
- 返回值为定时器的编号
- setTimeout可以传入不止两个参数,dalay后的参数作为回调函数的参数
```
setTimeout(function(a, b) {
  console.log(a+b)
}, 1000, 1, 2);
```
IE9以下只支持传入两个参数
-
