英语学好就可以很好理解JSON了(JavaScript Object Notation)

key,value形式，key必须为双引号扩起的字符串，value可以使string, number, object, array, null, true, false

JSON方法：
```js
JSON.stringfy(obj)  // 将js对象转化为json字符串
JSON.parse(JSONstring) //将json字符串转化为js对象
```
